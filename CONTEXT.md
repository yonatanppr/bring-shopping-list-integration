# Domain Glossary

**Shopping List**
: A named Bring collection shared with one or more Bring accounts.

**Default List**
: The shopping list used when a request does not name another accessible list.

**Integration Test List**
: The dedicated shopping list named `Integration Test` that may contain disposable items
  created by live contract tests. It is never a normal shopping list or the default list.

**Purchase Item**
: An item currently waiting to be purchased on a shopping list.

**Recently Used Item**
: An item no longer waiting to be purchased but retained by Bring for later reuse.

**Specification**
: A quantity, size, brand, or qualifier attached to an item without changing its name.

**Complete**
: Move a purchase item to recently used items.

**Remove**
: Permanently delete an item from a shopping list. Removal is distinct from completion.

**Mutation Receipt**
: A structured record that identifies a requested list change and the items confirmed by
  that request.
